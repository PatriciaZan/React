# ⚛️ React Mastery: Do Zero ao Full Projects

Bem-vindo ao repositório **PatriciaZan/React**. Este é um guia prático completo, organizado e progressivo para o ecossistema React. Desde fundamentos essenciais e Hooks avançados até projetos robustos integrados com TypeScript e bibliotecas modernas.

---

## 🚀 Estrutura do Repositório

O projeto é dividido em módulos estratégicos para facilitar o aprendizado:

### 1. `1-YT-IntroductionCourse/1Resumes`
**O Coração do React.** Resumos detalhados e desafios práticos focados na API central do React.
- **Hooks:** `useState`, `useEffect`, `useRef`, `useReducer`, `useId`, `useContext`.
- **Conceitos:** Passing Props, Portals, Custom Hooks e Regras de JSX.
- **Challenges:** Desafios de lógica com listas de compras, perfis e formulários.

### 2. `2-BeginnersProjects`
**Prática Real.** Pequenos projetos que consolidam o uso de estados e estilização.
- 🧮 Calculadora, Accordion, To-do List.
- 🛒 **E-Commerce Minimalista:** Filtragem por categorias, cores e preços.
- 🥗 Meals API e Testemunhos de usuários.

### 3. `3-WithTypeScript`
**Tipagem e Segurança.** Migração da lógica React para o mundo TypeScript.
- Tipagem de Props e Eventos.
- Hooks tipados (`useReducer` com actions fortes).
- Interfaces reutilizáveis e Context API com TS.

### 4. `4-withTypeScriptProjects` & `5-librarys`
**Workflow Moderno.** Integração com ferramentas do ecossistema e produtividade.
- Configurações avançadas com **Vite**, **Tailwind CSS** e **PostCSS**.
- Exploração de bibliotecas externas para otimização de UI e Gerenciamento de Estado.

## 📦 Como Instalar e Rodar

Como o repositório contém múltiplos projetos sub-divididos, cada pasta possui seu próprio `package.json`.

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/PatriciaZan/React.git
   ```

2. **Acesse o diretório do desafio ou projeto desejado:**
   ```bash
   cd 1-YT-IntroductionCourse/2BeginnersProjects
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   # ou
   npm i
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

## 📖 Destaques do Aprendizado

*   **Custom Hooks:** Criação de lógica reutilizável fora dos componentes.
*   **Context API:** Gerenciamento de estado global sem "prop drilling".
*   **Conditional Rendering:** Lógica de exibição de componentes (Weather, UserStatus).
*   **Ref Management:** Manipulação direta de DOM e timers com `useRef`.
*   **TypeScript Avançado:** Criação de `types.ts` para centralizar interfaces de usuários e produtos.
